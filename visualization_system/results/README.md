# Results Directory

This folder stores classification results with feature importance scores.

Generated by: `python mainClassify.py --num-events X`

## Contents

- `claude_classified_results_detailed.json` - Main results file with:
  - Event classifications
  - Confidence scores
  - Feature importance analysis (0-100 for each feature)
  - Event data snapshots
  - AI reasoning

## JSON Structure

```json
[
  {
    "event_id": 1,
    "true_label": "WIMP-like",
    "event_data": {
      "recoil_energy_keV": 5.249,
      "s2_over_s1_ratio": 3.024,
      ...
    },
    "api_analysis": {
      "classification": "WIMP-like (NR)",
      "confidence": 0.95,
      "reasoning": "..."
    },
    "importance_analysis": {
      "s2_s1_ratio_importance": 95,
      "energy_importance": 80,
      "pulse_shape_importance": 65,
      "position_importance": 45,
      "drift_time_importance": 40,
      "summary_reasoning": "..."
    }
  }
]
```

This data is used by `visualizer.py` to create charts.
