# Charts Output Directory

All visualization charts are saved here.

Generated by: `python visualizer.py`

## Main Charts (Always Generated)

1. **01_discrimination_bands.png**
   - S2/S1 vs Energy scatter plot
   - Physics classification regions
   - Color-coded by particle type
   - Resolution: 300 DPI

2. **02_class_distribution.png**
   - Bar chart of dataset composition
   - Event counts and percentages
   - Resolution: 300 DPI

3. **03_classification_accuracy.png**
   - Classification performance metrics
   - Confidence distribution
   - Only if classification results exist
   - Resolution: 300 DPI

4. **04_energy_distributions.png**
   - Energy spectra by particle type
   - Overlapping histograms
   - Resolution: 300 DPI

5. **05_s2s1_distributions.png**
   - S2/S1 ratio distributions
   - Linear and Log scale
   - Classification bands marked
   - Resolution: 300 DPI

## Feature Importance Charts (Optional)

- **03_importance_chart_X.png** (per classified event)
  - Feature importance bar chart
  - Only if `claude_classified_results_detailed.json` exists

## Summary Report

- **visualization_summary.txt**
  - Dataset statistics
  - Feature statistics
  - Classification metrics
  - Text format for easy review

---

**All charts are publication-quality PNG files at 300 DPI**
